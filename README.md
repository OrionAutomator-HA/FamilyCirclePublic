# Family Circle — public site

Official public pages for the **Family Circle** Android app (`com.familycircle.app`).

**Live site:** https://orionautomator-ha.github.io/FamilyCirclePublic/

## About the app

Family Circle is a shared family calendar and household helper. Sign in with Google, create or join a private family circle, and use shared calendars, notes, recipes, meal plans, and shopping lists with people you invite.
